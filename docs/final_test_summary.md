## 9. 附录

### 9.1 运行命令参考

```bash
# 运行所有测试
pytest

# 运行单元测试
pytest tests/unit/ -v

# 运行UI测试
pytest tests/unit/ui/ -v

# 运行E2E测试（排除真实设备测试）
pytest tests/e2e/ -v -m "not real_device"

# 运行设备测试（需要真实设备）
set REAL_DEVICE_TEST=1
pytest tests/e2e/test_device_initializer.py -v

# 生成覆盖率报告
pytest --cov=src --cov-report=html
```

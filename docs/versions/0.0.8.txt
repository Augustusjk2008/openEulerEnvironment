feat(protocol): 新增RESERVED字段类型支持

新增RESERVED协议字段类型，支持在原始帧中占位但不参与解析和日志输出。主要变更包括：
- 在协议模式定义中添加RESERVED类型选项
- 实现字段验证、布局计算和代码生成逻辑
- 更新协议编辑器UI以支持RESERVED字段配置
- 添加单元测试覆盖新功能

RESERVED字段特性：
- 可指定任意正数字节长度
- 有效时参与原始帧布局和解析帧结构体
- 无效时仅参与原始帧布局
- 不参与unpack赋值和schema构建

Release Date: 2026-04-19

fix(release): 发布 1.0.1 补丁版本

本次 1.0.1 版本是在 1.0 首次公开发布后的补丁更新，重点是修正公开仓库的 CI 发布基线：

- 将登录界面、主窗口导航、设置界面三组 Qt 界面测试明确标记为 GUI 测试
- 保持公共 GitHub Actions 基线只验证非 GUI、非设备依赖的公开单元测试
- 使 `master` 分支上的公开 CI 基线重新恢复为绿色

版本与发布信息同步更新：

- 将应用版本、首页版本标识、用户手册和仓库说明统一提升到 v1.0.1
- 新增补丁版本说明，保留 v1.0 作为历史公开版本记录

公共验证范围说明：

- 公共 CI 继续运行非 GUI、非 Ubuntu VM、非真实设备依赖的单元测试
- GUI 测试仍保留在仓库中，但应在具备 Qt 图形测试条件时单独执行

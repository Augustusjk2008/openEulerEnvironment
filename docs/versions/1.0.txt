Release Date: 2026-04-19

chore(release): 发布 1.0 公开版本

本次 1.0 版本聚焦于首个公开 GitHub 发布，统一了仓库对外信息与版本标识，主要包括：

- 将应用版本、首页展示版本、用户手册和仓库说明统一提升到 v1.0
- 补充公开版本说明，便于在教程页和仓库中查看当前发布记录
- 保持公开仓库以 PyInstaller 为唯一当前打包路径，避免公开文档与脚本出现多套打包口径
- 保留已有环境部署、SSH/SFTP、协议编辑、算法编辑和 SLOG 可视化能力作为 1.0 基线

公开发布基线说明：

- 当前公共 CI 仍以非 GUI、非设备依赖的单元测试为主
- 集成测试、GUI 测试与真机测试仍需要额外环境后再执行
- 仓库继续面向 Windows 桌面场景维护
